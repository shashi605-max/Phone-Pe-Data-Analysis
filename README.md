# 📱 PhonePe Executive Dashboard

This is a Power BI dashboard project created to analyze PhonePe transaction performance, user activity, and service-wise business insights.

## Dashboard Overview

The dashboard provides insights into:

- Total Transaction Amount
- Total Transaction Count
- Total Users
- Success Rate
- Monthly Transaction Growth (MOM%)
- Service-wise Transaction Analysis
- Top 5 Services by Transaction Amount
- Bottom 5 Services by Transaction Amount

- Platform maintains a 96% transaction success rate.
- Monthly transaction trends help identify growth and decline periods.
- Service-wise analysis highlights top-performing and low-performing services.
